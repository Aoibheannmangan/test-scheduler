RebookingForm(param0) => JSX.Element

A form component that allows user to reschedule and event Users can select a new start and end time, and provide a reason for the no-show

Kind: global function
Returns: JSX.Element - The JSX for rendering the rebooking form

Param Type Description
param0 Object The props object
param0.event Object The event to be rebooked, with details such as start and end time
param0.onSave function A function to save the updated event with new details
param0.onCancel function A function to cancel the rebooking process and close the form
