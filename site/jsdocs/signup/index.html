SignUp() => JSX.Element

SignUp component that renders the SignUp form and handles user registration

Validates user input, checks for existing users, creates new user accounts, navigates to the calendar page on success, displays alert if failed

Kind: global function
Returns: JSX.Element - The SignUp component
Component:
Example
