LogIn() JSX.Element

LogIn component that renders the LogIn form and handles user authentication

Checks for stored users, verifies credentials, navigates to the calendar page on success, displays alert if failed

Kind: global function
Returns: JSX.Element - The LogIn component
Component:
Example 

<Route path="/login" element={<LogIn />} />

LogIn~handleLogin(e) void

Handles the login form submission

Checks if the email and password submitted by the user match an already created account. Displays a success alert and navigates to the calendar page if credentials are correct. Displays an error alert if credentials are incorrect.

Kind: inner method of LogIn

Param Type Description
e React.FormEvent.<HTMLFormElement> The form submission event.
