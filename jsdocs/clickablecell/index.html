ClickableDateCell(param0) => JSX.Element

A custom clickable cell that allows the selection of a date unless it's been blocked

Kind: global function
Returns: JSX.Element - The code that renders the clickable date cell.

Param Type Description
param0 Object The prop object passed to the ClickableCell component
param0.children React.ReactNode The content to be displayed inside the date cell
param0.value Date THe date value the cell represents
param0.onSelectedSlot function A callback function that is triggered when the date cell is selected
param0.blockedDates Array.<Object> An array containing all the blocked dates
